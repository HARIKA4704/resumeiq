# ResumeIQ

Professional ATS Resume Analyzer built with Streamlit.

## Features

* ATS Score Analysis
* Resume Health Check
* Skill Gap Detection
* Resume Enhancement Suggestions
* PDF Resume Upload
* Keyword Matching
* Report Generation

## Tech Stack

* Python
* Streamlit
* Plotly
* Scikit-Learn
* PyMuPDF

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
